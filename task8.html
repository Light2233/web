Работа с модальными и немодальными формами

Теоретическая справка:

  • Модальная форма — блокирует взаимодействие с основной формой до своего закрытия. Используется для критических операций (ShowDialog() в C#).
  • Немодальная форма — позволяет работать с основной формой параллельно. Используется для не критичных операций (Show() в C#).
  • MenuStrip — компонент для создания главного меню приложения, обеспечивает навигацию.
